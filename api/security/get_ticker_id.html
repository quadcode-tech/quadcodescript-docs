get_ticker_id¶

get_ticker_id(ticker)¶
Parameters

  • ticker (string) – Input ticker.

Returns

integer

Retrieves the ID of the given ticker. If the ticker is not found - the function returns nil.

Example:

local ticker_id = get_ticker_id("EURUSD")
