rma¶

rma(series, period)¶
Parameters

  • source (series) – Input series

  • period (numeric) – MA period

Returns

series

Returns the rolling moving average of source for the period bars.

\[\mathrm{rma} = \frac{\mathrm{(\mathrm{period} - 1)\mathrm{rma}_1 + \mathrm{series}_0}}{\mathrm{period}}\]
