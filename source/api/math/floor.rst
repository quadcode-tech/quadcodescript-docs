floor
=====

.. qcs:function:: floor(x)
    :category: math

    :param x: Input value
    :type x: :qcs:type:`numeric` or :qcs:type:`series[numeric]`
    :returns: :qcs:type:`integer` or :qcs:type:`series[integer]`

    Returns the closest to the postitive infinity integer value, 
    that is not greater than **x**
