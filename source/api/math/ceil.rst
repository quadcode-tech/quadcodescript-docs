ceil
====

.. qcs:function:: ceil(x)
    :category: math

    :param x: Input value
    :type x: :qcs:type:`numeric` or :qcs:type:`series[numeric]`
    :returns: :qcs:type:`integer` or :qcs:type:`series[integer]`

    Returns the closest to the negative infinity integer value, 
    that is not less than **x**
