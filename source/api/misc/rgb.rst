rgb
===

.. qcs:function:: rgb(r, g, b)
    :category: misc

    :param r: The red component of the color in the range of :math:`[0,255]`
    :type r: :qcs:type:`integer`

    :param g: The green component of the color in the range of :math:`[0,255]`
    :type g: :qcs:type:`integer`

    :param b: The blue component of the color in the range of :math:`[0,255]`
    :type b: :qcs:type:`integer`

    :returns: :qcs:type:`color`

    This function will create a color from its individual components.
