drop_plot_value
===============

.. qcs:function:: drop_plot_value(name, bar_id)
    :category: misc

    :param name: Name of plot
    :type name: :qcs:type:`string`

    :param bar_id: An absolute bar index
    :type bar_id: :qcs:type:`integer`

    Sets the plot `name` value at index `bar_id` to **nan**. Current bar index is `current_bar_id`.
